# Nusantara Sakti Sales

Official Sales & CRM Platform for Nusantara Sakti Group. Specialized for Honda Vehicle Sales Management.

## Features

- **Dashboard**: Real-time sales performance and lead distribution analytics.
- **Lead Management**: Efficiently track and manage potential customers.
- **Sales Pipeline**: Visual drag-and-drop pipeline for deal management.
- **Broadcast System**: Integrated WhatsApp and Email marketing campaigns.
- **Task Management**: Structured tasks and reminders for sales teams.
- **Bilingual Support**: Full support for English and Indonesian languages.

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS + Shadcn UI
- Supabase (Backend & Database)
- i18next (Internationalization)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
